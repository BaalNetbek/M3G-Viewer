Getting Started

The M3G Viewer technology is designed for application developers who need to browse the content of a .m3g file. This technology is structured into two modules, the M3G Viewer and the M3G Toolkit. The M3G Viewer is the browser, a GUI-based tool for inspecting the data in a .m3g file. The M3G Viewer is built upon the M3G Toolkit which implements much of the Mobile 3D Graphics specification and provides the foundation for managing the .m3g file I/O.

This document assumes that the reader is already familiar with the Mobile 3D Graphics specification. It also assumes that the reader has a working knowledge of how the .m3g file is structured. If more information is required, then the reader may find the information discussed in this link to be useful.

To begin using the M3G Viewer, please proceed to the next page for instructions on how to load and view the contents of your .m3g file.
